## 📋 Descripción del Proyecto
Este proyecto realiza un análisis exploratorio de datos (EDA) sobre el dataset de **Telecom X**. El objetivo principal es identificar los factores críticos que influyen en la deserción de clientes (Churn) para proponer estrategias de retención basadas en datos.

## 🛠️ Tecnologías y Librerías
Para ejecutar este análisis, se utilizaron las siguientes herramientas:
* **Pandas:** Manipulación y limpieza de datos.
* **Requests:** Extracción de datos desde la API.
* **Matplotlib & Seaborn:** Visualización de patrones y comportamientos.
* **JSON Normalize:** Procesamiento de estructuras de datos anidadas.
